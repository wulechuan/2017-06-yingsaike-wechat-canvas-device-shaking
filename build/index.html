大数据开发工程师

岗位职责

  1. 负责公司大数据平台搭建、功能设计、及核心模块的开发；
  2. 负责基于大数据平台的数据仓库建设与数据应用开发；
  3. 负责为海量数据的处理和分析提供高效解决方案；
  4. 负责基于大数据平台搭建流式数据处理方案；
  5. 根据项目需求按时完成产相关产品的代码设计、开发和单元测试工作。

任职资格

  1. 计算机、数学或者统计学相关专业本科以上学历；
  2. 3年以上的Hadoop大数据平台开发经验，深入了解Hadoop数据处理体系（Hadoop/HDFS/Hive/YARN/HBase/Sqoop/Spark/Zookeeper/Kafka）；
  3. 具有丰富的数据仓库开发经验，了解数据仓库体系相关的技术；
  4. 了解大数据的基本处理思路与常用算法，技术基础扎实；至少熟悉使用2种编程语言（Java/Shell/Python/Scala），精通Hive SQL；
  5. 熟悉Linux操作系统；
  6. 优秀的问题分析与解决能力，条理逻辑清晰；具备良好的分析、组织、沟通能力；
  7. 热爱开源技术，关注并了解最新的大数据前沿技术；
  8. 良好的抗压能力与团队合作精神。

大数据科学家

岗位职责

  1. 负责数据挖掘分析模型规划设计，机器学习相关的大数据技术架构规划；
  2. 负责产品和用户的数据建模，提供预测分析结果，撰写相关分析报告；
  3. 推进数据模型的调研、开发、上线和使用，并进行模型的完善与优化；
  4. 探索数据挖掘专项项目，评估项目可行性，引导项目实施。

任职资格

  1. 本科以上学历，计算机、数学、统计学等相关专业，3年以上数据挖掘经验；
  2. 拥有完整的数据挖掘项目经验；
  3. 精通常用机器学习算法（如逻辑回归、聚类、神经网络、决策树、贝叶斯等）;
  4. 良好的机器学习原理和最优化理论基础;
  5. 熟练掌握SQL/HIVE，Python/R，熟悉Oracle数据库;
  6. 掌握Hadoop、Spark等分布式框架者优先；
  7. 拥有UBI项目经验者优先。

大数据运维工程师

岗位职责

  1. 负责大数据平台/Hadoop/storm/spark/OLAP等在线和离线数据平台运维保障；
  2. 负责大数据自动化运维以及数据化运营平台开发工作；
  3. 负责大数据平台架构审核、业务监控、持续交付、应急响应、容量规划等；
  4. 深入理解数据平台架构，发现并解决重大故障及性能瓶颈，打造一流的数据平台；
  5. 负责CDH各类大数据组件Hadoop、Hbase、Hive、Impala、Spark、ZK等安装调试和配置；
  6. 持续的创新和优化能力，提升产品整体质量，改善用户体验，控制系统成本

任职资格

  1. 计算机、数学或者统计学相关专业本科以上学历；
  2. 有2年以上Hadoop相关运维经验，了解Hadoop各组件的原理，并有实际部署维护经验；
  3. 深入理解linux系统，运维体系结构，精于容量规划、架构设计、性能优化；
  4. 熟悉Hadoop大数据生态圈，包括但不限于CDH、HDFS、YARN、Hive、HBase、Spark等；
  5. 精通一门以上脚本语言(shell/perl/python等)；
  6. 故障排查能力，有很好的技术敏感度和风险识别能力；
  7. 良好的服务意识，善于团队协作，主动思考，自我驱动力强；
  8. 熟悉Cloudera Hadoop的管理和维护大数据平台经验；
  9. 拥有Oracle 数据库运维经验优先。
